# Instagram Reels & Carousels — Full Specs

For edge cases when SKILL.md specs are insufficient.

## Reels (full)

| Param | Value |
|-------|-------|
| Native size | 1080×1920px (9:16) |
| Alt aspect | 1080×1440 (3:4) |
| Feed crop | 1080×1350 (4:5) |
| Min duration | 3s |
| Max duration | 180s |
| Formats | MP4, MOV |
| Codec | H.264, 30fps (60fps ok) |
| Audio | AAC, 48kHz stereo |
| Max file size | 4GB |
| Safe zone top | 15% |
| Safe zone bottom | 10% |

## Carousels (full)

| Param | Value |
|-------|-------|
| Portrait (best) | 1080×1350 (4:5) |
| Square | 1080×1080 (1:1) |
| Tall | 1080×1440 (3:4) |
| Landscape | 1080×566 (1.91:1) |
| Max slides | 10 |
| Consistency | All slides same size (first slide sets crop) |
| Formats | JPG, PNG |

## Caption best practices

- First ~125 chars visible in feed (rest behind "more")
- Line breaks every 2–3 lines for readability
- 80–120 words typical for reels; 50–80 for carousels
- CTA at end (link in bio, save, follow, comment)

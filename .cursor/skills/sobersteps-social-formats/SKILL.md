## Guardrails (non-negotiable)
- The profile does **not** sell the app. The profile tells the truth about the road.
- No clinical/therapeutic language.
- No promises/guarantees/outcomes (“quit in 30 days”, etc.).
- Don’t say “buy PRO” / “kup PRO”.
- No “VoidWalk” in the context of the app.

## Default workflow (fast)
If the user did not specify, assume:
- Platform: IG + TikTok friendly
- Length: 25–35s
- Output: hook (0–3s), middle (3–25s), close+CTA (25–35s), caption + hashtags

First, clarify internally (do not interrogate):
- What module is this about (Craving Surf / 3 AM / Mirror Moment / Return to Self / milestones / journal / Wall of Strength)?
- Which week/angle (awareness/distance/repair/integration)?
- What’s the “truth” (not a pitch) we’re saying?

## The triad must be present
Use as structure:
- Hook = **Uśmiech**
- Middle = **Perspektywa**
- CTA/close = **Droga**

```
←— Uśmiech ↔ Perspektywa ↔ Droga —→
```

## Formats (A–E)
### Format A — “Prawda o [temat]”
- Hook (0–3s): unexpected line
- Middle (3–20s): 3 concrete points
- Close (20–30s): one triad sentence + CTA

### Format B — “Mirror Moment”
- Hook: “3 minuty. Tylko ty i lustro.”
- Middle: show the moment / prompt
- Close: “Nie musisz lubić tego co widzisz. Wystarczy zostać.”

### Format C — “Noc 3 AM”
- Hook: night aesthetic, silence, title “3:00”
- Middle: “3 buttons” concept (breath / write / call)
- Close: “Wróć rano i powiedz że dałeś radę.”

### Format D — “80% wystarczy”
- Hook: “Przestań czekać aż będziesz gotowy.”
- Middle: “80% is enough” journaling frame
- Close: “Twoje 20% to już ruch.”

### Format E — “Tydzień [X] na ścieżce”
- Hook: “Tydzień [1/2/3/4]. Oto co mnie zaskoczyło.”
- Middle: one honest insight per week
- Close: “Następny krok jest prostszy niż myślisz.”
